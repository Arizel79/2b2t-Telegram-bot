# 2b2t Telegram bot
Telegram bot for getting statistics about players of the 2b2t server
The bot is available at the [link](t.me/lolritterbot)

## How to run
1. Clone this repo
2. Install dependents:
```
pip install -r requirements.txt
```
3. Run:
```
python3 main.py
```
